# mtsp-hill-climbing
<h3>
  <br>
mvn clean package
    <br>
java -jar target/mTSP.jar -d 5 -s 2 -v
    <br>
-d is depots count
    <br>
-s is salesman count
    <br>
-v is show with city names or show with city no
    <br>

  - -
  If you want to run hill climbing iteration just with inital random solution algorithm use :   <br>
  java -jar target/mTSP.jar -d 2 -s 5 -n random -v
